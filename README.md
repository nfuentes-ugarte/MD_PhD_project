# Molecular Dynamics Templates

Collection of input files and analysis scripts used during my PhD research in Molecular Biology at the University of Chile.

## Included Workflows

### Constant pH Molecular Dynamics (CpHMD)

* System preparation
* Energy minimization
* Heating (NVT)
* Equilibration (NPT)
* Production simulations (NVT)

### Classical Molecular Dynamics

* Energy minimization
* Heating (NVT)
* Equilibration (NPT)
* Production simulations (NPT)
* Trajectory analysis

### Hybrid Simulations

* QM/MM molecular dynamics
* Adaptive String Method

### Cluster Environment

* SLURM templates
* MPI execution examples

## Software

* AMBER24
* AmberTools24
* cpptraj
* Python 3
